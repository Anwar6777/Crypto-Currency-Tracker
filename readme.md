# 💰 Cryptocurrency Tracker

A modern and responsive cryptocurrency dashboard built using **HTML, CSS, JavaScript, Python (Flask)**, and the **CoinGecko API**. The application allows users to search for cryptocurrencies and view real-time market data, price trends, and key statistics through an interactive dashboard with both Light and Dark themes.

---

## ✨ Features

* 🔍 Search cryptocurrencies by name
* 💲 Live cryptocurrency price updates
* 📈 7-day price trend chart
* 🏆 Market rank display
* 💹 Market capitalization
* 📊 24-hour trading volume
* 🪙 Circulating supply
* 📦 Maximum supply
* 🚀 All-Time High (ATH)
* 📉 All-Time Low (ATL)
* 🌗 Light & Dark theme support
* 🔥 Top 10 cryptocurrencies dashboard
* 📅 Last updated timestamp
* ⚡ Fast and responsive user interface
* ❌ Error handling for invalid cryptocurrency searches

## 🚀 Installation

### 1. Clone the repository

```bash
git clone https://github.com/your-username/Crypto-Tracker.git
cd Crypto-Tracker
```

### 2. Create a virtual environment

```bash
python -m venv venv
```

### 3. Activate the virtual environment

**Windows**

```bash
venv\Scripts\activate
```

**macOS/Linux**

```bash
source venv/bin/activate
```

### 4. Install dependencies

```bash
pip install -r requirements.txt
```

### 5. Create a `.env` file

```env
COINGECKO_API_KEY=your_api_key_here
```

### 6. Run the Flask server

```bash
python app.py
```

Open your browser and visit:

```
http://127.0.0.1:5000
```
